Web Development

Development goes beyond just coding a working website. Every developer has the responsibility of upholding practices that keep the Web accessible, progressive, and relevant. Every website I create meets the following criteria:

  • Compliant with W3C (World Wide Web Consortium) standards
  • Cross-browser compatible
  • Mobile responsive
  • Fully accessible to individuals with disabilities
  • Optimized for Search Engines (SEO)
  • Troubleshooting, debugging, and quality control
  • Optional monthly maintenance
